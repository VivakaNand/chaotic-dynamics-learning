They are committed here so figures and leaderboards can be reproduced without re-training.

## Column definitions

**Model evaluation CSVs (`*_models_eval_results.csv`):**
- `Method`: model variant name
- `Tier`: Tier-1 (100k training steps) or Tier-2 (10k training steps)
- `Training`: Single-step, Curriculum K=1-16, or Ridge (no BP)
- `MSE`: mean squared error on the test trajectory
- `RMSE`: root mean squared error
- `R2`: coefficient of determination R²

**D2 CSVs (`*_d2_results.csv`):**
- `Model`: model variant name
- `D2`: estimated Grassberger-Procaccia correlation dimension from 5000-step rollout
- `D2_err`: absolute error |D2 − D2_true|

**IC robustness CSVs (`*_manual_ic_results.csv`, `*_rand_ic_results.csv`):**
- `IC`: initial condition (scalar for MG/Duffing, list string for Lorenz/KS)
- `Method`: model name
- `MSE`, `RMSE`, `R2`: metrics on the trajectory starting from that IC
